## 动态规划

C++中的动态规划（Dynamic Programming, DP）是一种通过分解问题、存储中间结果，从而减少重复计算、提高效率的算法技巧。DP主要适用于那些可以通过“子问题”的解逐步构建出“大问题”解的问题。
