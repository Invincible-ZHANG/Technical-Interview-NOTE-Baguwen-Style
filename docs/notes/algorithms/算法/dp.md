# c++动态规划

## 动态规划的基本概念

动态规划通常用于优化解决以下类型的问题：
* 重叠子问题：问题可以分解为多个子问题，且这些子问题会被多次计算。
* 最优子结构：问题的最优解可以通过子问题的最优解来构建。


为了避免重复计算，动态规划会使用一个数组或表格存储已经计算过的子问题的解，从而实现“记忆化”效果，降低时间复杂度。
